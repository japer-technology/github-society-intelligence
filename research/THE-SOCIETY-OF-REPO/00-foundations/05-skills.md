# Skills Catalogue

A Society of Repo operates across six operational domains. Skills are constitutional capabilities, not prompts.

---

## Skill domains

| Domain | Description | Key roles |
| --- | --- | --- |
| Internet operations | Searching, browsing, monitoring feeds and APIs | `web-research-bee` |
| Local document management | Indexing, organising, retrieving local files | `document-index-bee`, `intake-bee` |
| Software research and development | Code review, dependency analysis, build monitoring | `code-review-bee`, `dependency-bee`, `build-monitor-bee` |
| Business operations | Contracts, tax, staff, suppliers, finance | `contract-bee`, `tax-bee`, `staff-bee`, `supplier-bee`, `finance-watch`, `owner-briefing` |
| Personal operations | Scheduling, tasks, notes, reminders | `calendar-bee`, `task-bee` |
| Meta-administration | Observe and redesign the ecology itself | `activation-steward`, `memory-steward`, `representation-steward`, `evaluation-steward`, `ecology-monitor` |

---

## Assembly and meta-admin roles

Some roles exist to manage scale rather than to do frontline work.

| Role | Function | Default authority |
| --- | --- | --- |
| `assembly-bee` | Compresses lower-level outputs into assembly summaries before settlement | propose |
| `directive-bee` | Breaks approved settlements into narrower downstream tasks | act |
| `activation-steward` | Reviews routing quality, congestion, and K-line coverage | propose |
| `memory-steward` | Reviews memory temperature, drift, and recall quality | propose |
| `representation-steward` | Reviews whether durable records are stored in the right class | propose |
| `evaluation-steward` | Reviews metrics, credit assignment, and bootstrap fairness | propose |
| `ecology-monitor` | Reviews groupthink, objection quality, and society-level health | propose |

Meta-admin roles work from summaries by default and may propose structural changes, but may not merge governance changes unilaterally.

## Domain notes

### Internet operations
Source quality, staleness, privacy, and cloud-egress checks remain mandatory.

### Local document management
Retrieval results must preserve provenance and relational links back to source records.

### Software research and development
Build and code-review skills now feed dialogical quality metrics: objection usefulness, revision quality, and unnecessary deliberation rate.

### Business operations
High-impact proposals must cite relevant frames, procedures, prior decisions, and self-ideals.

### Personal operations
Summary-first routing is especially important because private context should surface in the smallest necessary form.

### Meta-administration
These roles watch society-level indicators:
- activation congestion
- memory drift
- representation conflicts
- bootstrap bias against new agencies
- objection usefulness and groupthink incidence

## Skills and the cognitive loop

Every skill domain runs through the same governed loop:

```text
stimulus
  → perception
  → frame selection
  → activation (K-lines, analogy, budgets)
  → agency or assembly response
  → criticism and inhibition
  → settlement
  → action
  → memory and credit assignment
  → reinforcement and ecology review
```

---

## Adding new skills

To add a new skill:

1. Define the operational context and dominant frames.
2. Propose bounded worker agencies.
3. Define required critics, censors, and insulation boundaries.
4. Draft the initial frame, K-line, and budget expectations.
5. Define evaluation, bootstrap protection, and credit-assignment expectations.
6. Submit as a governance PR.
