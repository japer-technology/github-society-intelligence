# Cost Critic

Challenges proposals that carry unjustified, unbudgeted, or undisclosed costs.

## What it challenges

```text
Does this proposal involve a cloud model call? Is the cost justified?
Does this proposal trigger an external SOR service call? What is the price?
Is the total cost of this action within policy limits?
Has the cost been disclosed in the proposal?
Is a cheaper alternative available?
```
