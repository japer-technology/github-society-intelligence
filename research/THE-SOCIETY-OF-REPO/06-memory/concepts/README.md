# Concept Candidates

A learning society should invent better abstractions, not only reinforce old routes.

Concept candidates are governance-reviewed proposals for new frames, semantic categories, comparison dimensions, or recurring labels.

---

## Concept candidate schema

```yaml
id: concept.{name}
proposed_by: agency-or-role-id
examples:
  - artifact-id
non_examples:
  - artifact-id
predicted_use:
  - text
governance_status: proposed | approved | rejected
linked_frames:
  - frame-id
```

Promotion requires examples, non-examples, predicted use, and governance review.
